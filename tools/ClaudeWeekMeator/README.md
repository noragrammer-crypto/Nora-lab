# Claude週制限トラッカー

Claudeの週制限の残量を管理するシンプルなWebアプリです。

## 機能

- 📊 現在の使用率を入力して残量を表示
- ⏰ 次回リセット（カスタマイズ可能な曜日と時刻）までの残り時間をリアルタイム表示
- 📈 今日使える推奨枠を自動計算（残量 ÷ 残り日数）
- 💾 使用率と設定をブラウザに自動保存
- 🔧 リセット曜日と時刻を自由に設定可能

## 使い方

1. ブラウザで `index.html` を開く
2. 現在の使用率（%）を入力
3. リセット曜日を選択（デフォルト：金曜日）
4. リセット時刻を選択（デフォルト：19:00）
5. 「更新」ボタンをクリック
6. 残量と今日使える枠が表示されます

## リセットタイミング

デフォルトは日本時間の毎週金曜日 19:00 ですが、アプリ内で自由にカスタマイズできます。

## 技術仕様

- Pure HTML/CSS/JavaScript（ライブラリ不要）
- LocalStorageで使用率を保存
- レスポンシブデザイン対応
