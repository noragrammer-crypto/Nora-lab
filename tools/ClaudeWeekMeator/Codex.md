# Codex Guide: ClaudeWeekMeator

このディレクトリ（ClaudeWeekMeator）を変更する際の Codex 向け作業メモです。

## 目的
- 変更対象をこのディレクトリ内に限定して安全に作業する。
- 影響範囲に応じた最小テストを確実に実行する。

## 作業ルール
1. 変更前に関連ドキュメント（README / CLAUDE.md / spec / docs）を確認する。
2. 変更はこのディレクトリに直接関連するものを優先する。
3. テストコマンドがある場合は必ず実行し、結果を記録する。

## テスト
- `package.json` がある場合: scripts を確認して必要なテストを実行。
- `__tests__/` がある場合: 変更範囲に応じて unit / functional / e2e を選択。
- テストが未整備の場合: 最低限、実行確認手順を PR に記載する。

## 仕上げ
- 変更理由・影響範囲・確認内容を簡潔にまとめる。
- 無関係なファイルが変更されていないか確認する。
